# Практическая работа №1
## Цели:
Для любого документа или набора документов  написать токенизатор, провести нормализацию полученного словаря (стемминг, лемматизация, синонимия, стоп-слова и т.п.) и построить простой анализатор тональности текста на естественном языке (на основе метода VADER-правил или наивного байесовского классификатора).

Программную реализация необходимо разместить в любом открытом GIT-репозитории. В файле README необходимо привести инструкцию по сборке, развертыванию и использованию анализатора с примерами работы анализатора тональности, в том числе с пользовательскими данными. Альтернативно - создать архив проекта, разместить в облачном хранилище. Допустимые форматы: zip, tar.gz, 7z.

## Задачи:
- Выбрать набор данных;
- Провести нормализацию данных;
- Построить простой анализатор тональности текста;
- В файле README необходимо привести инструкцию по сборке, развертыванию и использованию анализатора с примерами работы анализатора тональности, в том числе с пользовательскими данными. 
- Разместить программную реализацию в GIT-репозитории.

## Решение:
Файл ```main.ipynb``` содержит в себе программную реализацию, выполняющую работу с текстами песен группы "Imagine Dragons". Так как набор данных уже отчищен, то выполняется частичная предобработка текста, включающая в себя нормализацию словарей и разработка анализатора текстов.

Файл ```main.py``` является основным и создан на основе ```main.ipynb```. Данный файл работает с текстовым файлом, расположенным в ```data\text.txt```. Что касается операций с тестом, то также выполняется отчистка и анализ тональности текста. В качестве результат выводится: указанный текст, оценка по каждому из критериев и итоговое заключение.

*Пример:*

```
Указанный текст:
I really love eating dumpling.

Анализ тональности:
{'neg': 0.194, 'neu': 0.248, 'pos': 0.558, 'compound': 0.6032}

Итог:
Ваш текст - позитивный
```

## Инструкция по сборке
- Клонировать данный репозиторий командой: ```git clone https://github.com/so1ed/MoNLP_lab.git```
- В python окружении запустить: ```pip install -r requirements.txt```
- Указать в файле ```text.txt``` необходимый текст.
- Выполнить команду: ```python main.py```
